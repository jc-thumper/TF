Merge Duplicate Data.

Key Features:

  • Allows you to merge the duplicate data.
  • User can merge the duplicate data of any model like Product, Partners, Product Category as well as any Custom model.
  • It will update the duplicate record reference with original record at all places in your odoo system.
  • It gives the option for what action you want to perform on the duplicate record like Delete, Archived or Nothing.
See Video

Overview

  • Let's see how this app is work. For eg. I have Duplicate record into the Partner Table, and the partner table reference is exist at Sales order, Invoices and many more places.
  • Duplicate record has the Sales orders data.
  • Duplicate record has the Invoice data.
  • For merge the Original and Duplicate record, go to Menu Settings > Custom Shortcuts. While you click on the menu Merge Duplicate Data, one pop-up window will open.
  • In opened pop-up window, you have to select Model, then Original record and Duplicate record. Also select action option for duplicate record. After click on Merge Data button, It will replace the reference of the Duplicate record with Original record at all the places in odoo system.
  • In Sales order, the duplicate record is update with original record.
  • In Invoices, the duplicate record is update with original record.
  • As Action option select, the duplicate record is deleted from selected model.
